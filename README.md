# 📊 Sales Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing sales data to uncover trends, measure business performance, and generate actionable insights using an interactive Power BI dashboard. The goal is to help businesses make data-driven decisions to improve revenue and efficiency.

---

## 🎯 Business Problem

Businesses often struggle to track sales performance across regions, products, and time periods. This project aims to:

* Analyze overall sales performance
* Identify top-performing products and regions
* Track monthly and yearly sales trends
* Improve decision-making through visual insights

---

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning & Transformation (Power Query)
* Data Visualization
* DAX (Data Analysis Expressions)

---

## 📁 Dataset Information

* Sales transactions dataset
* Includes:

  * Order Date
  * Region
  * Product Category
  * Sales Amount
  * Profit
  * Quantity Sold

---

## 🧹 Data Cleaning & Preparation

* Removed missing and inconsistent values
* Transformed data using Power Query
* Created calculated columns and measures using DAX
* Ensured proper data types and relationships

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Sales
* Total Profit
* Total Orders
* Profit Margin

### 🔹 Visual Analysis

* Sales trend over time (monthly/yearly)
* Region-wise sales performance
* Category-wise and product-wise analysis
* Top and bottom performing products

### 🔹 Interactive Filters

* Region filter
* Time period filter
* Product category filter

---

## 📈 Key Insights

* Certain regions contribute significantly higher revenue
* A small number of products generate the majority of sales (Pareto principle)
* Seasonal trends affect sales performance
* Some categories have high sales but low profit margins

---

## 💡 Recommendations

* Focus on high-performing regions to maximize revenue
* Improve pricing strategy for low-profit products
* Optimize inventory based on sales trends
* Promote top-performing products

---

## 🚀 Project Outcome

This dashboard enables stakeholders to monitor business performance in real-time and make informed strategic decisions to increase sales and profitability.
